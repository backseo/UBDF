Compétences en développement.

Etudiant au Wagon de Bordeaux (2017).
Une expérience intensive et d'une richesse indéniable tant dans la diversité des profils que l'on y rencontre que dans le contenu qu'elle délivre. J'ai pu parfaire mes connaissances en Front-end et découvrir le code coté Back-end. Je découvre le framework Ruby on Rails, l'architecture MVC, la programmation orientée objets, les classes, l'héritage, les modules, les librairies, les helpers, les requêtes HTTP, le CRUD, les bases de données, la collaboration sur les projets...
MERCI à toute l'équipe de Bordeaux ! Et big-up au Batch #102 !
